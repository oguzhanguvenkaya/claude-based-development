# /profile-update — Skills Inventory Update

## Usage
```
/profile-update
```

## Purpose
Review recent wiki activity across all domains and update `_meta/profile.md` to reflect current skill levels, new competencies, and growth areas.

## Steps

### 1. Scan Recent Activity
- Read `log.md` (global) and each domain's `log.md`
- Identify: new pages created, topics covered, queries answered
- Note which domains have been most active

### 2. Assess Growth
- Compare current wiki coverage against previous profile
- Identify new skills or deepened understanding
- Note areas that haven't been touched recently

### 3. Update Profile
- Edit `_meta/profile.md` with:
  - New skills added
  - Skill level changes (learning → active → strong)
  - New projects or achievements
  - Updated date

### 4. Report
- Present a brief "what changed" summary
- Suggest areas for next focus based on goals
